<h2>『KU Info Village』- 大学の授業情報掲示板</h2>
<br>
<p>開発期間：10ヵ月(リファクタ含む)<br>開発人数：1名<br>ユーザー数：約600名(2022/08/15時点)<br>※一般の方はご利用いただけません。また、一部公開していないファイルがあります、ご了承ください。</p>


<p>簡易紹介：<br>　開発者が通う大学の授業情報掲示板、通称『KU Info Village』というWebアプリケーション。各学生がこれまでに受けた授業の感想や単位取得難易度に関する情報を5段階の評価と共に投稿することで、次の履修登録で参考にし合うというコンセプト。</p>
